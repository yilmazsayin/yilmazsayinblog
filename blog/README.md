# Blog taslağı

Bu klasör statik bir blog taslağıdır (HTML + CSS + küçük JS).

## Çalıştırma

- `blog/index.html` dosyasını tarayıcıda aç.
- Yazı örneği: `blog/post.html`

## Özelleştirme

- Renkler/typography: `blog/styles.css` (`:root` değişkenleri)
- Yazı listesi (taslak): `blog/script.js` içindeki `POSTS`
